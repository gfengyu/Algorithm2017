# Algorithm2017
2017秋算法课程实验

ex1: 直接插入排序，堆排序，归并排序，快速排序  
ex2: 冒泡排序，快速排序，基数排序，计数排序

## project 2
ex1: 矩阵链乘 - 动态规划法

## project 3

支持操作

- 构建顺序统计树、查找顺序统计树第i小的关键字
- 输出树的所有信息

## project 4

ex2: 求所有点对的最短路径 - 无向图上的Johnson算法
